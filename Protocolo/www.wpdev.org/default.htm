Welcome to Wolfpack Wiki

Wolfpack is an Ultima Online(tm) server software.

There also exists an Wolfpack Client.

Wolfpack is open source, free software licensed under the GNU General Public License.

Wolfpack's Mission

  • Cross Platform -- Runs under both Windows and Unix/Linux
  • Extensible Features -- Provides high levels of configuration / customization
  • Community Focused -- Prompt support, experienced developers, pleasant attitude
  • Open Source -- Open development process, anyone can participate
  • Feature Rich -- See List of Features
  • Current -- Stay on top of recent changes
