loop
{
    uint Item_Serial
    ushort Artwork
    byte Layer
    ushort Hue
} 		This is a loop of all equipped items on the character.

Item_Serial: The serial of the item.
Artwork: The item's artwork.
    - Add the flag 0x8000 if the item's hue is not zero.
Layer: The item's clothing layer.
Hue: The item's hue. Include this only if Artwork includes the flag 0x8000.
uint 0x00000000 Signifies the end of the item loop.
