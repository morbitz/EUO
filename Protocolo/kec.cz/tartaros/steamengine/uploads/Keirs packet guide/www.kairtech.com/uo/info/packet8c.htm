Ultima Online Protocol :: Server Redirect
Packet # Packet Name Length Client Server Obsolete
0x8C Server Redirect 0x000B X X
Data Type Description
byte 0x8C The packet number.
uint Address The server's IP address.
ushort Port The server's port.
uint Key The gameplay encryption key. This is usually the same as the account number.

Packet Description:
This is sent to redirect a client to a gameserver after the user selects a shard.
