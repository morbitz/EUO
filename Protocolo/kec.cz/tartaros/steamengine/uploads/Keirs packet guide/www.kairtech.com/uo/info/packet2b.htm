Ultima Online Protocol :: God Mode
Packet # Packet Name Length Client Server Obsolete
0x2B God Mode 0x0002 X
Data Type Description
byte 0x2B The packet number.
bool enabled If true, god mode is enabled and access to god client features is allowed.

Packet Description:
This packet is sent by the server to either enable or disable god mode.

Send this packet after receiving a Request God Mode packet.
