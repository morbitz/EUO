loop
{
    uint Serial
    ushort Artwork
    ushort Hue
    ushort Amount
    uint Price
    byte Length
    char[] Name
} 		This loop contains information for each item in the shop inventory.

Serial: The item's serial.
Artwork: The item's artwork number.
Hue: The item's hue.
Amount: The amount of the item available.
Price: The price of the item.
Length: The length of the item's name.
Name: The name of the item.

Packet Description:
This is sent to display the player's shop inventory when selling to a vendor.
